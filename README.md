# Python Keylogger (Educational)

This project demonstrates a simple keylogger written in Python for cybersecurity learning purposes.

## Features
- Records keyboard input
- Saves keystrokes to a file
- Uses the pynput library

## Installation

Install the required library:

pip install pynput

## How to Run

Run the program:

python keylogger.py

The keystrokes will be saved in:

keylog.txt
